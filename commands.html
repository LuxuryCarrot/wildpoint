⚡ 서버 명령어

Wild-Point 서버에서 사용할 수 있는 모든 명령어를 확인하세요

📋 명령어 가이드

🏠 이동 관련

/spawn 스폰 지점으로 이동
/home [홈이름] 설정한 집으로 이동
/sethome [홈이름] 현재 위치를 집으로 설정
/delhome [홈이름] 집 삭제
/back 이전 위치로 돌아가기
/warp [워프이름] 워프 지점으로 이동
/tpa [플레이어] 플레이어에게 이동 요청
/tpaccept 이동 요청 수락

💰 경제 시스템

/balance 또는 /돈 잔액 확인
/baltop 부자 순위 확인
/pay [플레이어] [금액] 돈 송금
/worth 손에 든 아이템 가격 확인

🏪 상점 시스템

/상점 메인 상점 메뉴
/서버상점 서버 공식 상점
/광물상인 광물 상점
/농작물상인 농작물 상점
/블럭상인 블럭 상점
/인테리어상인 인테리어 상점

🛒 유저 거래소

/유저거래소 열기 유저 거래소 열기
/유저거래소 등록 [가격] 아이템 등록 (손에 든 아이템)
/유저거래소 보관함 보관함 확인

🎣 낚시 시스템

/낚시 낚시 메뉴
/낚시터 낚시터로 이동
/낚싯대상인 낚싯대 구매
/복합낚싯대 복합 낚싯대 제작

🎲 미니게임

/주사위 [금액] 주사위 게임 참여
/룰렛 [금액] 룰렛 게임
/주식 주식 메뉴

💬 소통 관련

/msg [플레이어] [메시지] 개인 메시지
/r [메시지] 마지막 메시지에 답장
/ignore [플레이어] 플레이어 차단/해제
/mail send [플레이어] [메시지] 메일 전송

🎒 인벤토리 관리

/kit [키트이름] 키트 수령
/kits 사용 가능한 키트 목록
/창고 창고 열기
/쓰레기통 쓰레기통 열기

🎮 게임 기능

/메뉴 퀵 메뉴 열기
/야간투시 또는 /야투 야간투시 토글
/밥 허기 회복
/수리 아이템 수리
/제작대 제작대 열기
/야생 야생으로 이동

📖 도감 & 정보

/도감 도감 열기
/도감조회 [플레이어] 다른 플레이어 도감 확인
/list 온라인 플레이어 목록
/ping 핑 확인
/getpos 현재 좌표 확인
/디스코드 디스코드 정보

💡 사용 팁

• 권한 확인: 일부 명령어는 특정 권한이 필요할 수 있습니다

• 쿨다운: 대부분의 명령어에는 쿨다운이 적용됩니다

• 수수료: 일부 명령어 사용 시 수수료가 발생할 수 있습니다

• 도움말: /help 명령어로 추가 정보를 확인할 수 있습니다

• 한국어 지원: 대부분의 명령어가 한국어로 사용 가능합니다
