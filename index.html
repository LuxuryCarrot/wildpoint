🏠 Wild-Point 서버에 오신 것을 환영합니다!

Wild-Point는 다양한 커스텀 콘텐츠와 플러그인으로 가득한 프리미엄 마인크래프트 서버입니다. 커스텀 작물 재배부터 특별한 아이템까지, 바닐라와는 다른 특별한 경험을 제공합니다.

⚡ 서버 명령어

🌾 작물 관련 명령어

/crops info - 작물 정보 확인
/crops give [작물명] - 작물 씨앗 지급 (관리자)
/season - 현재 계절 확인

💰 경제 명령어

/money - 잔액 확인
/pay [플레이어] [금액] - 돈 송금
/shop - 상점 열기

🏠 기본 명령어

/spawn - 스폰 지점으로 이동
/home - 집으로 이동
/sethome - 집 설정
/tpa [플레이어] - 플레이어에게 텔레포트 요청

📋 서버 규칙

모든 플레이어가 즐겁게 게임할 수 있도록 다음 규칙을 준수해주세요.

  1. 욕설 및 비방 금지 - 다른 플레이어에 대한 욕설, 비방, 차별적 발언을 금합니다.
  2. 그리핑 금지 - 다른 플레이어의 건축물을 파괴하거나 훼손하는 행위를 금합니다.
  3. 핵/치트 프로그램 사용 금지 - X-Ray, 자동클릭 등 불공정한 프로그램 사용 시 영구 밴됩니다.
  4. 스팸 금지 - 채팅창에 같은 내용을 반복하거나 광고성 메시지를 금합니다.
  5. 개인정보 보호 - 다른 플레이어의 개인정보를 요구하거나 공유하지 마세요.
  6. 버그 악용 금지 - 서버나 플러그인의 버그를 악용하지 마세요. 발견 시 관리자에게 신고해주세요.
  7. AFK 농장 제한 - 과도한 AFK 농장은 서버 성능에 영향을 줄 수 있어 제한될 수 있습니다.
  8. 건축 가이드라인 - 공공장소에서는 적절한 크기와 디자인으로 건축해주세요.

🔧 서버 플러그인

Wild-Point 서버에서 사용 중인 주요 플러그인들을 소개합니다.
